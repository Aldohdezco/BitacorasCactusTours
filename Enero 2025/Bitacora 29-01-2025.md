# Reunión con Luis David: Implementación del Algoritmo de Distribución de Vehículos  

**1. Reunión con Luis David**  
- Durante la reunión, se abordaron varios puntos clave relacionados con la implementación del algoritmo de distribución de vehículos.  
- **Explicación sobre el Funcionamiento de la Empresa:**  
  - Luis David explicó cómo trabaja la empresa en cuanto a la asignación de vehículos y la distribución de unidades.  
  - Se detalló cómo está estructurada la base de datos (BD) para manejar las distribuciones de vehículos, los vehículos disponibles y las asignaciones de los servicios.  

**2. Ejercicio de Consulta de Distribución de Vehículos**  
- Luis David me asignó un ejercicio práctico para realizar una consulta que obtuviera las distribuciones de un vehículo específico desde la base de datos.  
  - El ejercicio incluyó la estructuración correcta de la consulta y la obtención de datos relevantes sobre las asignaciones de vehículos.

**3. Programación del Algoritmo de Distribución de Vehículos**  
- Se comenzó con la programación del algoritmo que optimiza la distribución de vehículos según ciertos parámetros y reglas predefinidas.  
  - El algoritmo considera variables como la disponibilidad del vehículo, la distancia, y otros factores críticos.  

**4. Desarrollo de Alertas para Casos Específicos**  
- Me dediqué a crear alertas para gestionar ciertos casos específicos durante la asignación de vehículos.  
  - **Caso Específico:** Se implementó una alerta para cuando no se acepta una persona para asignar una Unidad de Transporte Vehicular (UTV), alertando sobre el error o la limitación del sistema.
