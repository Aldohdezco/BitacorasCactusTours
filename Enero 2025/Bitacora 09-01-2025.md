# Avances en el Desarrollo del Módulo de Administración de DropOffs
- **Modelos para la Creación de DropOffs**:
  - Se desarrollo el modal necesario para gestionar la creación de DropOffs en el sistema.

- **Modal para la Alerta de Confirmación de Eliminación**:
  - Se diseñó e implementó un modal para confirmar la eliminación de DropOffs.
  - El modal incluye:
    - Mensaje personalizado para informar al usuario sobre la acción a realizar.
    - Botones de confirmación y cancelación para manejar la interacción del usuario.
