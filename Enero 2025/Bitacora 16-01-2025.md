# Proyecto de Automatización de Envío de Correos

- **Implementación en PHP para la Automatización del Envío de Correos**:
  - Se desarrolló un script en PHP que utiliza la API de ZeptoMail para automatizar el envío de correos electrónicos.
  - Funcionalidades implementadas:
    - Configuración de las credenciales de la API de ZeptoMail.
    - Creación de una función para enviar correos electrónicos con parámetros personalizables (destinatarios, asunto, cuerpo del correo).
    - Manejo de respuestas de la API para verificar el éxito o los errores en los envíos.

- **Pruebas Iniciales**:
  - Se realizaron pruebas básicas para verificar el correcto funcionamiento del script y la conexión con la API.
  - Se enviaron correos de prueba exitosamente.
