# Segunda Reunión con Luis David

Se realizó una segunda reunión por la tarde en la que se abarcaron los siguientes temas:

- **Funcionamiento detallado de la aplicación**:
  - **Index (Login)**:
    - Proporciona una interfaz para la autenticación de los usuarios.
    - Validación de credenciales en el backend.
    - Redirección al "main" en caso de éxito.

  - **Main**:
    - Sirve como la página principal tras el inicio de sesión.
    - Contiene los componentes principales de la aplicación.
    - Acceso a funcionalidades clave a través de botones y menús.

  - **Estructura de Layouts**:
    - **Main**:
      - Hamburguer para navegación.
      - Espacio principal para las funcionalidades dinámicas.
      - Footer de navegacion. 

- **Uso de Firebase**:
  - **Autenticación**:
    - Gestión de usuarios a través de Firebase Authentication.
    - Manejo de distintos proveedores de inicio de sesión (correo electrónico, Google, etc.).
  
  - **Base de Datos**:
    - Uso de Firestore como base de datos en tiempo real para almacenar y sincronizar datos.
    - Estructuración de colecciones y documentos para optimizar consultas.

  - **Configuración**:
    - Configuración de permisos y reglas de seguridad en la consola de Firebase.
