# Avances en Curso de React y Conexión al Servidor


- **Continuación del Curso de React con Vite.js**:

- **Inicio de Pruebas de Conexión al Servidor**:
  - Se realizaron pruebas iniciales para establecer la conexión del widget al servidor.
  - Actividades realizadas:
    - Configuración de un servidor básico.
    - Implementación de endpoints para probar el envío y recepción de datos.
    - Pruebas desde el widget para consumir los endpoints.
  - **Resultados**:
    - Las pruebas **no tuvieron éxito** debido a problemas identificados en:
