# Creación de un Manual de Usuario para las Interfaces de DropOffs

- Se elaboró un manual dirigido al usuario final, explicando el uso de las interfaces relacionadas con el módulo de DropOffs.
- El manual incluye:
- **Guía de Uso General**: Instrucciones paso a paso para navegar y utilizar las funcionalidades de DropOffs.
- **Creación de DropOffs**: Detalles sobre cómo registrar un nuevo DropOff.
- **Listado de DropOffs**: Explicación sobre cómo visualizar la lista de DropOffs y sus estados.
- **Edición y Eliminación**: Procedimientos para editar o eliminar un DropOff existente.
- **Cambio de Estado Automático**: Información sobre cómo se gestionan los cambios de estado y eliminación automática después de 10 minutos.
