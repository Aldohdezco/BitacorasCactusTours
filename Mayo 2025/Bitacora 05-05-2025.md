# Actividades Realizadas

- **Reconstrucción de la interfaz de actividades**: Se inició el proceso de reconstrucción total de la interfaz, comenzando específicamente con la sección de **paquetes**, ya que es el componente con mayores cambios y estructura más compleja.
- **Implementación progresiva**:
  - Se desarrolló el **constructor** de paquetes para estructurar correctamente sus propiedades.
  - Se diseñaron las **rows** necesarias para mostrar los paquetes en la interfaz con un formato claro y dinámico.
  - Se comenzó a implementar la lógica de las **funcionalidades conectadas a la base de datos**, sentando las bases para la interacción entre interfaz y datos reales.
